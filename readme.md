# Описание проекта

## 1. Общее описание

Этот проект состоит из двух основных файлов: **`shell_emulator.py`** и **`test_shell_emulator.py`**.

Файл **`shell_emulator.py`** реализует оболочку терминала, которая эмулирует взаимодействие с файловой системой, хранящейся внутри ZIP-архива. Пользователь может выполнять команды для навигации по архиву, удаления файлов и папок, а также для отображения истории команд. Основные команды включают `ls`, `cd`, `rm`, `rmdir`, `history` и другие.

Файл **`test_shell_emulator.py`** содержит набор тестов для проверки функциональности команд, реализованных в `shell_emulator.py`. Тесты проверяют корректную работу таких команд, как `ls`, `cd`, `rm`, `rmdir` и вывод истории команд.

## 2. Описание всех функций и настроек

### Файл `shell_emulator.py`:

- **`ShellEmulator.__init__(config_path, silent=False)`**  
  Инициализирует эмулятор файловой системы, загружает конфигурацию из TOML-файла, текущий каталог устанавливается в корневой, и выполняется стартовый скрипт, если он присутствует.

- **`load_config(config_path)`**  
  Загружает конфигурацию из указанного TOML-файла, включая имя компьютера, путь к ZIP-архиву и путь к стартовому скрипту.

- **`load_virtual_fs()`**  
  Загружает виртуальную файловую систему из ZIP-архива и сохраняет список файлов для дальнейшей обработки.

- **`run_startup_script()`**  
  Выполняет команды из стартового скрипта, если он существует в архиве.

- **`execute_command(command)`**  
  Выполняет переданную команду, поддерживает такие команды, как `ls`, `cd`, `rm`, `rmdir`, `history`, и выводит результат.

- **`ls()`**  
  Выводит список содержимого текущей директории. Показывает файлы и папки, находящиеся в ZIP-архиве.

- **`cd(path)`**  
  Меняет текущую директорию на указанную. Поддерживается переход в родительские и поддиректории.

- **`rm(path)`**  
  Удаляет указанный файл из текущей директории, если он существует в архиве.

- **`rmdir(path)`**  
  Удаляет указанную директорию и её содержимое, если директория существует.

- **`show_history()`**  
  Выводит историю всех команд, введенных пользователем.

- **`exit_shell()`**  
  Завершает работу оболочки и закрывает ZIP-архив.

- **`show_output(output)`**  
  Выводит текст на консоль, если режим **silent** не активирован.

### Файл `test_shell_emulator.py`:

Тесты обеспечивают проверку функциональности команд оболочки.

- **`test_shell_emulator()`**  
  Основная тестовая функция, которая инициализирует эмулятор с подавлением вывода (silent=True) и запускает несколько тестов:
  
  - **Test 1**: Проверка вывода содержимого директории (`ls`).
  - **Test 2**: Проверка вывода содержимого несуществующей директории.
  - **Test 3**: Тест команды `cd` с переходом в существующую директорию.
  - **Test 4**: Тест команды `cd` с несуществующей директорией.
  - **Test 5**: Проверка вывода истории команд.
  - **Test 6**: Проверка пустой истории команд.
  - **Test 7**: Удаление существующего файла с помощью команды `rm`.
  - **Test 8**: Попытка удалить несуществующий файл.
  - **Test 9**: Удаление существующей директории с помощью команды `rmdir`.
  - **Test 10**: Попытка удалить несуществующую директорию.

Все тесты проверяют корректность выполнения команд и выводят сообщение **"All functions executed successfully!"** при успешном прохождении всех тестов.
