# Визуализатор зависимостей Node.js

## Общее описание
Этот проект реализует визуализатор зависимостей для пакетов Node.js. Он извлекает файл `package.json` из ZIP-архива, собирает зависимости и визуализирует их в графическом формате с использованием Mermaid. Это позволяет разработчикам легко увидеть и понять структуру зависимостей их проектов.

## Структура файлов
Проект состоит из следующих файлов:
- `visualize-dependencies.js`: Содержит реализацию функций для извлечения зависимостей, генерации графа и визуализации.
- `visualize-dependencies.test.js`: Содержит тесты для проверки функциональности.
- `config.json`: Содержит параметры конфигурации, такие как путь к ZIP-архиву и выходной путь для сохранения графа.
- `package.json`: Определяет зависимости проекта и необходимые скрипты для запуска.

## Функциональность
Визуализатор зависимостей поддерживает следующие функции:
- Извлечение `package.json` из ZIP-архива.
- Получение зависимостей из `package.json`.
- Нормализация версий зависимостей.
- Рекурсивный сбор всех зависимостей, включая подзависимости.
- Генерация графа зависимостей в формате Mermaid.
- Визуализация графа и сохранение его в файл.

## Описание функций

### extractPackageJson
Извлекает и парсит `package.json` из указанного ZIP-архива.

#### Параметры:
- `archivePath`: Путь к ZIP-архиву.

#### Возвращает:
- Объект JSON, представляющий `package.json`.

### getDependencies
Возвращает зависимости из `package.json`.

#### Параметры:
- `packageJson`: Объект JSON `package.json`.

#### Возвращает:
- Объект с зависимостями.

### normalizeVersion
Нормализует версию, извлекая последнюю стабильную.

#### Параметры:
- `version`: Строка с версией.

#### Возвращает:
- Нормализованную версию или 'latest'.

### collectDependencies
Рекурсивно собирает все зависимости, включая подзависимости.

#### Параметры:
- `dependencies`: Объект с зависимостями.
- `allDependencies`: Объект для накопления всех зависимостей.
- `visited`: Множество для отслеживания посещенных пакетов.

#### Возвращает:
- Объект со всеми собранными зависимостями.

### generateMermaidGraph
Генерирует граф зависимостей в формате Mermaid.

#### Параметры:
- `dependencies`: Объект со всеми зависимостями.

#### Возвращает:
- Строку, представляющую граф в формате Mermaid.

### visualizeGraph
Визуализирует граф и сохраняет его в файл.

#### Параметры:
- `graph`: Граф в формате Mermaid.
- `outputPath`: Путь для сохранения выходного файла.
- `MermaidGraphPath`: Путь к временно сохраняемому графу Mermaid.

### Запуск визуализатора
Чтобы запустить визуализатор зависимостей, выполните следующую команду:

```bash
node visualize-dependencies.js
```
